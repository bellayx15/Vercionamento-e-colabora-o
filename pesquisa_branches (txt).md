&#x20;Uma branch é uma cópia separada de um projeto, que deixa você adicionar, editar ou fazer testes de modo que não altere o projeto principal. É muito útil para o desenvolvimento pois com ela  é possível fazer várias alterações, como manter os códigos organizados (os incompletos não afetam o trabalho), permite que diversas pessoas mecham em partes diferentes ao mesmo tempo sem dar problemas, o desenvolvedor pode fazer testes para testar o código e pode deletar a branch quando quiser, além disso é muito fácil realizar correções de erros de forma bem rápida.

&#x20;Como por exemplo: se uma falha no login é identificada, o desenvolvedor precisa arrumar o mais rápido possível. Se ele utilizar o branch pode corrigir a falha do login rapidamente e testa-la antes de enviar para ver se deu certo. Outro exemplo é quando uma equipe está trabalhando em um projeto importante e precisa acessar ele de diferentes computadores, com a a branch, os membros da equipe podem acessar o projeto de pcs diferentes e mexer em coisas diferentes dele, o que é muito útil. Um último exemplo é quando um membro de uma equipe realiza um código incompleto, como o uso da branch esse código não afetará os completos, pois ficará em uma área separada.
